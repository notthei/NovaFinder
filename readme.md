
# NovaFinder
macのspotlightは使いずらい！！！！！！fk

# 機能
- **/ + option**にて入力ボックスを表示します。
- **/shell** <ターミナルを開きます>
- **/open** <finderを開きます>
- **/calc** <簡単な計算ができます> useage-[/calc 1+1]

# 追加予定のコマンド・機能
- **/reboot** <再起動>
- **/shutdown** <シャットダウン>
- **/shell <任意のコマンド>**

- 任意のコマンドの設定[/setting(設定アプリを起動)など]をjsonで管理
- ai連携
- UI設定
#

常駐で起動してた方が便利なのでspotlightをoffにし以下の設定を行いましょう!

```
設定 -> 一般 -> ログイン項目と拡張機能 -> ログイン時に開く
```
**ログイン時に開く**にてアプリを追加したら起動時に実行されます。
